# Delivery Management

Three-pane dispatch console: queue (left), map (center), order detail + timeline (right). Designed for a desk dispatcher overseeing multiple riders.

## Screens
- `index.html` — Live dispatch view with stylized map (CSS-drawn, brand-colored pins)

## Notes
- Map is intentionally schematic — production would use Mapbox or similar.
- Pins use brand yellow (numbered stops), black (depot), green (done), red (alert).
