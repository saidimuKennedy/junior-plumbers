# Admin Dashboard

Multi-outlet operations overview. Owner / exec primary audience; store managers can drill into their own outlet.

## Screens
- `index.html` — Overview (today's KPIs, 14-day chart, alerts, outlets table)

## Components illustrated
- Top app bar with brand lockup
- Sidebar nav with grouped sections, active = black-fill yellow icon
- KPI cards with serif hero numbers
- Mock bar chart with brand-yellow + black bars
- Live alerts list (semantic dots + badges)
- Outlets performance table (progress bars, status badges)
